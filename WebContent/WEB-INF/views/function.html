VDP Customization and Production

VDP is short for virtual data product. In the registry, VDP is the same as physical data product. In the reality, VDP doesn't exist. Once a VDP is requested, a corresponding production workflow will be invoked and dynamically generate a physical data product. VDP is more flexible than traditional data products and allows modelers to customize all sorts of data to fit their models.

Geospatial Web Services

CyberConnector depends on web services. Most processes in generating VDP are conducted by web services. CyberConnector provides a registry to archive service metadata and enable users to search and use them in designing a new VDP.
