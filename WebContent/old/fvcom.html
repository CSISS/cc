River forcing file

River discharge data including number of rivers, discharge volume, discharge temperature, and discharge salinity. River discharge data are extracted from USGS website (http://waterdata.usgs.gov/usa/nwis/uv). The file is generated based on four fixed observation sites (the site no in the basket is defined by USGS):

  • Atchafalaya_River_at_Morgan_City (07381600)
  • Caernarvon_Outfall_Channel (295124089542100)
  • Mississippi_River_at_Belle_Chasse (07374525)
  • Wax_Lake_Outlet (07381590)

Respectively, the list of adopted river properties for each site is (parameter no is defined and explained by USGS):

  • 00065; 00060
  • 00065; 00010; 00060; 00095; 00480
  • 00065; 00055; 00060
  • 72020; 00060
