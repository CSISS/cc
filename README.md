# CyberConnector

NSF EarthCube Building Block project

This project aims to connect the existing data sources to the Earth science models so the generation of modeling products could be more automatic and effortless.

# Prerequisite

JDK 1.8+

Tomcat 8.0+

MySQL 5.5+

ncWMS 2.4.1+

PyCSW 2.3+

Maven 3.5.0+
